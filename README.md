# ☕ Coffee Shop Web Application

A full-stack **Coffee Shop Management System** built using **PHP, MySQL, HTML, and CSS**.  
This application allows users to browse products, manage a cart, place orders, and book reservations — simulating a real-world café system.

---

## 📌 Overview

This project demonstrates practical implementation of:

- User authentication systems  
- Session-based cart management  
- Order processing workflows  
- Database-driven web applications  
- Reservation systems  

It is designed as both a **learning project** and a **portfolio-ready application**.

---

## 🚀 Features

### 🔐 Authentication
- User registration and login
- Session management
- Logout functionality

### 🛒 Cart & Shopping
- Browse menu items
- Add/remove items from cart
- Dynamic cart updates
- Checkout process

### 📦 Order Management
- Place orders
- View order history
- Detailed order information

### 📅 Reservation System
- Book tables
- View personal reservations
- Manage reservation records

### 👤 User Profile
- Update user information
- Upload/change avatar
- Track personal activity

### 💳 Payment Support
- Payment method selection
- Improved checkout system

### 🖼️ Image Handling
- Product image support
- Avatar upload system
- Image fixing utility script

## 🔄 Application Flow
- User signs up or logs in
- Browses available menu items
- Adds items to cart
- Proceeds to checkout
- Order is stored in the database
- User can view orders and reservations
